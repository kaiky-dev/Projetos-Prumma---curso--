Tags Sêmanticas

O que são?

As tags semânticas em HTML são elementos que fornecem significado ao conteúdo da página, além de simplesmente definir sua apresentação visual. Essas tags ajudam a estruturar a página de uma maneira mais significativa, facilitando a compreensão do conteúdo por parte dos desenvolvedores e dos motores de busca.

Aqui estão algumas das principais tags semânticas em HTML5:

< header >

 
Representa o cabeçalho de uma seção ou página. Pode conter elementos como logotipos, títulos, menus de navegação, etc.
< nav >

 
Representa uma seção de navegação. Deve conter links ou outros elementos de navegação.
< main >

 
Representa o conteúdo principal da página. Geralmente, não inclui cabeçalhos, rodapés ou barras laterais.
< article >

 
Representa um conteúdo independente e autocontido, como um post de blog ou um artigo.
< section >

 
Representa uma seção tematicamente agrupada de conteúdo. Pode conter diversos tipos de conteúdo.
< aside >

 
Representa conteúdo relacionado ao conteúdo circundante, como barras laterais, notas ou anúncios.
< footer >

 
Representa o rodapé de uma seção ou página. Pode conter informações de copyright, links de navegação, etc.
