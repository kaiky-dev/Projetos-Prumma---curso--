Tags de formatação

O que são?

A expressão "tag de formatação" geralmente se refere a elementos HTML que são usados para controlar a apresentação e o estilo do texto em uma página web. Essas tags permitem que os desenvolvedores web definam a estrutura visual e a aparência do conteúdo.

Algumas delas são:


Negrito

< b >
 Usada para deixar o texto em negrito.

Enfatizado

< i >
 Usada para deixar o texto em itálico.

Marcação

< mark >
 Usada para deixar o texto marcado.

Pequeno

< small >
 Usada para deixar o texto pequeno.

Subscrito

< sub >
 Usada para deixar o texto subscrito.

Sobrescrito

< sup >
 Usada para deixar o texto sobrescrito.

Deletado / Riscado

< del >
 Usada para deixar o texto riscado.

Sublinhado

< ins >
 Usada para deixar o texto sublinhado.

Contêiner

< span >
 Contêiner para aplicar estilos específicos.
